使用前可以修改的选项

  1. $spacer 文本间距
  2. $enable-flex 是否默认伸缩布局
  3. $enable-rounded 是否默认圆角
  4. $enable-shadows 是否默认阴影
    5. QAQ
修改方法 Open the _variables.scss file find the $enable-flex variable. Change it from false to true. Recompile, and done! 修改完对bootstrap.scss进行编译即可 IE9之前不兼容伸缩布局flex bootst4使用grunt构建的
